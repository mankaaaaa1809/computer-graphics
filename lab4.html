Пошаговый алгоритм

Простейший алгоритм растеризации отрезка. Вычисляет координаты точек отрезка с использованием уравнения прямой y = kx + b. Шаг изменения координат зависит от угла наклона отрезка.

  • Использует вещественную арифметику
  • Простой в реализации
  • Медленнее целочисленных алгоритмов

Алгоритм ЦДА (Цифровой дифференциальный анализатор)

Алгоритм основан на вычислении приращений координат. На каждом шаге к текущим координатам добавляются вычисленные приращения dx и dy. Использует вещественную арифметику.

  • Более эффективен, чем пошаговый алгоритм
  • Использует вещественные числа
  • Требует округления координат
Время выполнения: 0 мс
Промежуточные вычисления:

Алгоритм Брезенхема для отрезков

Эффективный алгоритм, использующий только целочисленную арифметику. На каждом шаге выбирается следующая точка, наиболее близкая к идеальной прямой. Минимизирует ошибку растеризации.

  • Только целочисленные операции
  • Высокая скорость выполнения
  • Широко используется в графических библиотеках

Алгоритм Брезенхема для окружности

Алгоритм для растеризации окружности, использующий симметрию. Строит только одну восьмую часть окружности, а остальные точки получаются симметрией. Использует только целочисленные операции.

  • Использует симметрию окружности
  • Только целочисленные вычисления
  • Высокая эффективность

Алгоритм Ву (Xiaolin Wu's line algorithm)

Алгоритм сглаживания линий (антиалиасинг), предложенный Сяолинем Ву в 1991 году. Позволяет создавать визуально гладкие линии за счет использования градаций цвета пикселей.

  • Создает сглаженные линии без "лесенки"
  • Использует градации интенсивности цвета
  • Основан на вычислении расстояния до идеальной линии
  • Обрабатывает два пикселя на каждом шаге (основной и соседный)

Алгоритм Кастла-Питвея (через алгоритм Ву)

Модификация алгоритма Брезенхема для сглаживания линий, реализованная через алгоритм Ву. Сочетает эффективность целочисленных вычислений с качеством сглаживания.

  • Использует целочисленную арифметику Брезенхема
  • Применяет сглаживание алгоритма Ву
  • Эффективный компромисс между скоростью и качеством
  • Основан на вычислении ошибки и её распределении между пикселями
