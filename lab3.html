Рекомендации по тестированию

Для тестирования методов подобраны изображения:

  • С неравномерным освещением
  • С низкой контрастностью и шумом
  • Текст на текстурированном фоне
  • Изображения с импульсным шумом (для медианного фильтра)
Важная рекомендация: Для зашумлённых изображений сначала примените медианный фильтр с большим размером окна (до 39), затем адаптивную пороговую обработку.
Подсказка: Нелинейные фильтры особенно эффективны для предобработки изображений перед пороговой обработкой. Экспериментируйте с разными размерами фильтров!
