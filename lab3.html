Описание методов

Локальная пороговая обработка — определение порога для каждого пикселя на основе статистики локальной области (окна). Позволяет лучше обрабатывать изображения с неравномерным освещением.

Метод Ниблэка вычисляет порог как функцию от среднего и стандартного отклонения в окне.
Метод Савольды использует среднее значение и градиент для адаптивного выбора порога.

Адаптивная пороговая обработка — алгоритм, представленный в лекции, который динамически подстраивает порог для каждой области изображения, обеспечивая высокое качество бинаризации даже при сложных условиях освещения.

Нелинейные фильтры на основе порядковых статистик — фильтры, которые заменяют значение пикселя на статистику (медиану, минимум, максимум, квантиль) значений в окрестности. Эффективны для подавления шумов, подчеркивания деталей и предобработки перед пороговой обработкой.

Тестовые изображения

Для тестирования методов подобраны изображения:

  • С неравномерным освещением
  • С низкой контрастностью и шумом
  • Текст на текстурированном фоне
  • Изображения с импульсным шумом (для медианного фильтра)
Рекомендация: Для зашумлённых изображений сначала примените медианный фильтр, затем адаптивную пороговую обработку.
